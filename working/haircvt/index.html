HAIRCVT

HAIRCVT is a new service to help you make inspired choices with your hair: and then find the perfect stylist to cut it for you.

My Role

As sole developer, I was responsible for the build of HAIRCVT's full tech stack, which included:

  • the HAIRCVT iOS and Android apps
  • an admin area to manage salons, stylists and hairstyles
  • the HAIRCVT website
  • a private API to handle the sycning of data between the apps, database, and the website

The website is responsive, and uses a custom module factory to manage dependencies and bootstrap functionality. The History/PushSate API is used with the filters to maintain state and upload search results in situ.

The Node.js applications are deployed to Amazon EC2 instances using Capistrano. Images are stored on Amazon S3.
